‚ùå **Don't:**
- Try to redesign everything at once
- Forget mobile responsiveness
- Skip testing on different screen sizes
- Clone without customizing for your audience
- Ignore accessibility standards
- Create inconsistent spacing/typography
- Make cards too large or too small
- Forget to add save/wishlist functionality
- Overcomplicate navigation
- Ignore user feedback

‚úÖ **Do:**
- Implement in phases
- Test extensively on mobile
- Keep consistent spacing (8px grid)
- Maintain typography hierarchy
- Follow accessibility guidelines
- Use consistent colors throughout
- Make cards scannable at a glance
- Always include secondary actions (save, share)
- Keep navigation simple and logical
- Get feedback early and often
