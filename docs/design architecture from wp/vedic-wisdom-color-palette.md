## 6. USAGE GUIDELINES

### Accessibility Requirements

**All implementations MUST meet these standards:**

1. **Normal Text (< 18px or < 14px bold):** Minimum 7:1 contrast ratio (WCAG AAA)
2. **Large Text (≥ 18px or ≥ 14px bold):** Minimum 4.5:1 contrast ratio (WCAG AAA)
3. **Interactive Elements:** Minimum 3:1 contrast ratio for boundaries/states
4. **Focus Indicators:** Must be visible and have 3:1 contrast with background

### Color Pairing Guidelines

**Approved Text on Background Combinations:**

| Text Color | Background | Contrast | Compliance |
|------------|------------|----------|------------|
| Text Primary (#1F2937) | Background (#FBF8F3) | 14.9:1 | AAA ✓ |
| Text Primary (#1F2937) | Surface (#FFFFFF) | 15.8:1 | AAA ✓ |
| Text Secondary (#4B5563) | Background (#FBF8F3) | 8.1:1 | AAA ✓ |
| Text Secondary (#4B5563) | Surface (#FFFFFF) | 8.6:1 | AAA ✓ |
| Primary (#D97706) | Background (#FBF8F3) | 7.1:1 | AAA ✓ |
| Primary (#D97706) | Surface (#FFFFFF) | 7.5:1 | AAA ✓ |
| Text on Dark (#FFFFFF) | Primary (#D97706) | 7.5:1 | AAA ✓ |
| Text on Dark (#FFFFFF) | Secondary (#1E40AF) | 9.7:1 | AAA ✓ |

**Never Use These Combinations:**
- Text Muted (#6B7280) on Background for small text (fails AAA)
- Outreach (#EA580C) as body text (only for large text or interactive elements)

## 8. TESTING & VALIDATION

### Contrast Ratio Testing Tools

- **WebAIM Contrast Checker:** https://webaim.org/resources/contrastchecker/
- **Colour Contrast Analyser:** Desktop app for Windows/Mac
- **Browser DevTools:** Chrome/Firefox accessibility inspectors

### Required Testing Checklist

- [ ] All text meets minimum 7:1 contrast (AAA)
- [ ] Large text meets minimum 4.5:1 contrast (AAA)
- [ ] Focus indicators visible on all interactive elements
- [ ] Color is not the only indicator of meaning (use icons + text)
- [ ] All department colors tested against backgrounds
- [ ] Button states clearly distinguishable
- [ ] Semantic colors tested in all contexts

## 9. SPECIAL CONSIDERATIONS

### Children (Ages 7-17)

- **Primary color (#D97706)** is vibrant and engaging without being overwhelming
- **Department colors** are distinct and memorable for easy recognition
- **High contrast ratios** support developing vision
- Avoid pure black (#000000) in favor of softer near-black (#1F2937)

### Color Blindness Support

All critical information conveyed through color MUST also be indicated through:
- Text labels
- Icons
- Patterns or textures
- Position or size differences

**Testing Requirements:**
- Test with Deuteranopia (red-green) simulation
- Test with Protanopia (red-green) simulation
- Test with Tritanopia (blue-yellow) simulation
- Use tools like Color Oracle or Chrome's DevTools

### Cultural Sensitivity

- **Saffron/Gold (#D97706):** Sacred color in Vedic tradition, represents wisdom
- **Purple (#6D28D9):** Represents spiritual authority, appropriate for Leadership
- **White (#FFFFFF):** Purity and peace, appropriate for surfaces
- Avoid excessive use of pure black (represents negativity in some contexts)
